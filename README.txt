Αυτό το αρχείο περιέχει οδηγίες για την σωστή λειτουργία του προγράμματος για 
την EshopDB καθώς και μερικές εξηγήσεις για το περιεχόμενο που παραδίδεται.


Για την ομαλή λειτουργία του προγράμματος θα πρέπει ΠΡΩΤΙΣΤΩΣ να εκτελεστούν τα δύο παρακάτω αρχεία:

< EshopDB_dump.sql > που περιέχει όλη την βάση δεδομένων μαζί με τα δεδομένα εισόδου ( αρχικά δεδομένα ).
< users.sql > που περιέχει τους αρχικούς χρήστες της βάσης.

Σε περίπτωση που δεν εκτελεστούν τα παραπάνω το πρόγραμμα θα εμφανίσει μήνυμα λάθους και θα τερματίσει.

	b)  Να ανοίξετε το αρχείο Eshop_Employee\src\Employee.java και να αλλάξετε την γραμμή < private static int Port = 3307; > θέτοντας 
	    ως Port αυτό που επιθυμείτε.

	                                       ΠΕΡΙΕΧΟΜΕΝΑ ΠΑΡΑΔΟΤΕΟΥ
								
				
	
	< Eshop_CustomerAndGuest > , Project που περιέχει όλα τα αρχεία java που χρειάζονται για να χρησιμοποιήσει την βάση μας ένας πελάτης ( Customer )
	                             αλλά και ένας Guest.
								 
								 
	< Eshop_Employee > , Project που περιέχει όλα τα αρχεία java που χρειάζονται για να χρησιμοποιήσει την βάση μας ένας υπάλληλος ( Employee ).
	
	
	< Excecutables for Windows > , φάκελος που περιέχει δύο αρχεία .jar ένα για τον υπάλληλο και ένα για τον Guest και τον πελάτη (δες παράγραφο ΕΚΤΕΛΕΣΙΜΑ ΑΡΧΕΙΑ ΑΠΟ WINDOWS ).
	
	< mysql-connector-java-8.0.13 > , αρχείο που περιέχει το jar το οποίο πρέπει να γίνει import και στα δύο παραπάνω projects ώστε να γίνει εφικτή 
	                                  η επικοινωνία του προγράμματος με το MySQL Workbench ( mysql-connector-java-8.0.13\mysql-connector-java-8.0.13.jar ).
	
	
	< EshopDB_dump > , αρχείο sql που περιέχει την βάση δεδομένων EshopDB
	
	< users.sql > , αρχείο sql που περιέχει του αρχικούς χρήστες της βάσης
	

	
	                                ΕΚΤΕΛΕΣΙΜΑ ΑΡΧΕΙΑ ΑΠΟ ΤΟ ΠΡΟΓΡΑΜΜΑ ECLIPSE
										   
										   
										   
	Στο Project < Eshop_CustomerAndGuest > το εκτελέσιμο αρχείο είναι το < Eshop_CustomerAndGuest\src\FirstScreen.java > ενώ τα υπόλοιπα αρχεία java 
	                                       βοηθούν στην σωστή εκτέλεση τους προγράμματος.
										   
										   
	Στο Project < Eshop_Employee > το εκτελέσιμο αρχείο είναι το < Eshop_Employee\src\Employee.java > ενώ τα υπόλοιπα αρχεία java 
	                                       βοηθούν στην σωστή εκτέλεση τους προγράμματος.									   
	
	
	                                      ΕΚΤΕΛΕΣΙΜΑ ΑΡΧΕΙΑ ΑΠΟ WINDOWS 
										  
										  
	Στο παραδοτέο παραδίδονται επίσης δύο αρχεία .jar , τα οποία είναι τα εξής :
	
	< Excecutables for Windows\Eshop_Employee.jar >
	< Excecutables for Windows\Eshop_CustomerAndGuest.jar >
	
	Με διπλό κλίκ σε αυτά τα αρχεία ( ή από αντίστοιχη εντολή σε command line ) το κάθε πρόγραμμα θα τρέξει χωρίς να χρειαστεί να ανοίξετε το πρόγραμμα Eclipse
	Το μεν πρώτο περιέχει την λειτουργία του υπαλλήλου ενώ το δεύτερο περιέχει την λειτουργία του Πελάτη και του Guest χρήστη.
	
	Σε αυτό το σημείο ΣΗΜΕΙΩΝΟΥΜΕ ότι τα παραπάνω δύο αρχεία έχουν φτιαχτεί για το Port 3307 , άρα σε περίπτωση διαφορετικού Port δε θα λειτουργήσουν
	( θα εμφανιστεί σχετικό μήνυμα ). Αν θέλετε να χρησιμοποιήσετε διαφορετικό Port τότε πρέπει να ανοίξετε τον Eclipse.
